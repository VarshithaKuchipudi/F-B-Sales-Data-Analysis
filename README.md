**Sales Analysis Dashboard (Excel)**

**Overview**
This project is an interactive Sales Analysis Dashboard created in Microsoft Excel to analyze and visualize sales performance. The dashboard provides valuable business insights through dynamic charts, KPI cards, and interactive filters, enabling users to monitor revenue trends, customer spending, and order performance efficiently.

**Objective**
The objective of this project is to transform raw sales data into meaningful insights that help businesses:
	Monitor overall sales performance.
	Identify top-performing products and categories.
	Analyze customer purchasing behavior.
	Track monthly revenue trends.
	Understand city-wise order distribution.
	Evaluate sales during different occasions.
	Identify peak order hours for better operational planning.
	
**Dashboard KPIs**
The dashboard displays the following Key Performance Indicators (KPIs):
		Total Orders: 1,000
		Total Revenue: ₹3,520,984
		Average Order-to-Delivery Time: 5.53 Days
		Average Customer Spend: ₹3,520.98
		
**Dashboard Visualizations**
Revenue by Categories
Displays revenue generated from different product categories such as:
		Cake
		Colors
		Raksha Bandhan
		Soft Toys
		Sweets
This visualization helps identify the highest revenue-generating category.

**Top 10 Cities by Orders**
Shows the top 10 cities based on the number of orders received, making it easier to identify high-demand regions.

**Revenue by Occasions**
Compares revenue generated during different occasions including:
		Anniversary
		Birthday
		Diwali
		Holi
		Raksha Bandhan
		Valentine's Day
This helps businesses understand seasonal demand and plan marketing campaigns accordingly.

**Revenue by Products**
Displays revenue generated by individual products such as:
		Desserunt Box
		Dolores Gift
		Harum Pack
		Magnam Set
		Quia Gift
This analysis highlights the best-selling products.

**Monthly Revenue Trend**
Illustrates revenue generated across different months, enabling users to identify seasonal sales patterns and business growth trends.

**Revenue by Order Time**
Shows revenue distribution based on the hour of the day, helping businesses determine peak ordering times and optimize staffing or promotional activities.

**Interactive Features**
The dashboard includes interactive filters to enhance data exploration:
**Occasion Slicer**– Filter dashboard metrics by specific occasions.
**Order Date Timeline**- Analyze sales performance for selected months or date ranges.

These filters update all charts and KPIs dynamically.
